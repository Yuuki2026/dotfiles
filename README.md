# dotfiles

My personal Linux desktop configuration.

## Environment

* OS: Arch Linux
* WM: Hyprland
* Bar: Waybar
* Terminal: Kitty
* Shell: Zsh
* Prompt: Starship
* File Manager: Thunar
* Input Method: Fcitx5

## Directory Structure

```text
home/
├── .config/
│   ├── hypr/
│   ├── waybar/
│   └── kitty/
└── .zshrc
```

## Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/dotfiles.git
```

Copy the files to your home directory:

```bash
cp -r home/. ~/
```

Or create symbolic links if you prefer to manage everything through Git.

## Features

* Modular Hyprland configuration
* Waybar configuration
* Kitty configuration
* Zsh + Starship prompt
